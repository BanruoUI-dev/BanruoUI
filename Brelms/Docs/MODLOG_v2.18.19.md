# Bre v2.18.20 - 输出动作抽屉（UI阶段）

## 目标
- 输出动作抽屉：先做“结果系统”的面板与交互骨架（不接入条件系统/不执行动作）。

## 本次变更
- 新增 Actions 页签的 UI 骨架：
  - 顶部“新增”按钮 + “动作类型”下拉框。
  - 动作类型（当前）：缓入缓出、旋转。
  - 动作条目列表：可折叠（+/-）、可拖拽重排（MouseUp 时确认顺序）。
  - 条目字段：启用勾选、名称可编辑（默认=动作类型）。
  - 条目参数（UI占位）：持续时间、延迟开始；旋转额外提供角度输入。
  - “条件”下拉框新增并默认提供“无条件”。
- 文案：将“新增 动作”中的“动作”更名为“动作类型”。
- 版本号：TOC 更新为 v2.18.19。

## 说明
- 仅面板/交互骨架：当前所有动作条目数据为会话态（不写 DB / 不执行动作）。
